# jungsunwoung_skt

##사용법


<ul>
  <li>1. 압축파일을 엽니다</li>
    <li>2. target 폴더에 들어갑니다</li>
    <li>3. java -jar jungsunwoung-0.0.1-SNAPSHOT.jar 입력</li>
    <li>4. API 콜을 보내봅니다</li>
  <li>API 콜은 http://localhost:80에서 진행합니다</li>
    <li></li>
</ul>
